Shopping Mall Management System (SQL)
Project Overview
This project manages a shopping mall database. It handles store rentals, space allocation, and tenant management.

What's inside?
Tables: Owners, Spaces, SpacesRental, StoreTypes, etc.

Logic: Advanced Triggers for automated calculations and Stored Procedures for mall operations.

Error Handling: Implemented to ensure data integrity during rentals.

Key Features
Automated rental status updates.

Email notification logic for tenants.

Comprehensive reporting on mall occupancy.
